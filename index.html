How to install
1
Download the zip file above and unzip it to a folder on your computer.
2
Open Chrome and go to chrome://extensions
3
Enable Developer mode using the toggle in the top-right corner.
4
Click Load unpacked and select the unzipped extension folder.
5
Click the extension icon in Chrome, enter your name and you're ready to log!
⚠ If Chrome says the extension is unverified, click Keep anyway — this is expected for internal tools not published to the Chrome Web Store.
